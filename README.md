## Flix-Api Project

Projeto desenvolvido na formação da pycodebr, usando Django e Django Rest Framwork com o intuito de construir uma API que atenda às mais diversas necessidades de consumo do backend construído. 

Projeto modularizado em apps segmentados de acordo com as funcionalidades necessárias (actors, genres, movies e reviews). 

Foi possível compreender melhor como estruturar um backend robusto que atenda a diversos consumidores de uma forma padronizada, usando as melhores práticas.
